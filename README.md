# SmartDoc-RAG

> Chat with your PDF documents using AI   
> Built using Retrieval-Augmented Generation (RAG), LangChain, HuggingFace embeddings, and ChromaDB.

---

# Overview

>SmartDoc-RAG is an intelligent document question-answering system that allows users to interact with PDF files conversationally.
>It processes documents, converts them into embeddings, stores them in a vector database, and retrieves relevant context to generate accurate answers using LLMs.

---

# Features

-  Load and process PDF documents  
-  Smart text chunking  
-  Semantic search using embeddings  
-  Context-aware answers using RAG  
-  Fast retrieval with ChromaDB  
-  HuggingFace / Mistral model support  
-  Secure API key management using `.env`  

---

# Tech Stack

- Python  
- LangChain  
- HuggingFace Embeddings  
- ChromaDB  
- PyPDFLoader
